# Code+Role

This folder defines delivery responsibilities, working flow, and push standards for each implementation role.

## Structure
- `Backend/README.md`: backend engineer guide and push rules.
- `Frontend/README.md`: frontend engineer guide and push rules.

## Team Rule
- Backend and Frontend work in separate branches by role scope.
- Every pull request must link to the corresponding FR issue (`FR-xxx`).
- Do not push directly to `main`.
